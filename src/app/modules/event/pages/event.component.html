What is a Private Event?

A Private Event is where a User creates a private/exclusive event where people can see it on the Chello platform.

Is a Private Event free on Chello?

A Private Event is not free on Chello. The User must charge Customers to view a Private Event. Only paid Customers can see the Private Event.

Can the User who created the Private Event share it for free?

Yes, after the Private Event is finished, the User can share the Private Event for free. We encourage the User to wait 14 days before the Private Event is shared for free. This will encourage people to want to pay for your Private Event.

What are some examples of a Private Event on Chello?

Examples of a Private Event are concerts, comedy shows, car shows, exclusive events, etc.

Does a User have to be a Creator to create a Private Event?

No, a User does not have to be a Creator to create a Private Event on Chello.

Can Adult Content be a Private Event?

NO! No adult content is allowed on creating a Private Event. To create adult content, you need to be a Creator on Chello.

What are the fees to create a Private Event on Chello?

Chello will charge $1 per ticket of the Private Event in addition to 6% of the total. Example, User creates a car show, selling tickets for customers to view on Chello. User charges $5 for this Private Event. 25,000 Customers pay for this Private Event. The cost that Chello will collect on the day of the event, three hours before the event starts, is $125,000. From this example, User will receive from this Private Event $92,500. Chello made $1 per ticket sold which was 25,000=$25,000 + the 6% of the $125,000=$7500. User will get the funds within 3 business days after the Private Event.
